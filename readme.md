# SaaS Scheduling Platform

Sistema SaaS para gerenciamento de agendamentos de serviços, desenvolvido com arquitetura em camadas, suporte a múltiplos estabelecimentos e preparado para escalar.

O projeto simula uma plataforma onde diferentes negócios podem cadastrar serviços e receber agendamentos de clientes através de páginas personalizadas.

---

## Tecnologias

### Frontend
- React
- TypeScript
- Vite
- CSS

### Backend
- Node.js
- Express
- TypeScript
- Prisma ORM

### Banco de dados
- PostgreSQL

### Storage
- Supabase Storage

---

## Arquitetura

O projeto segue uma arquitetura em camadas, separando frontend e backend.
